\Blender-Final

AI-powered Blender Python (BPY) Code Generator built using Qwen 2.5 Coder, LoRA Fine-Tuning, FastAPI, React, and PostgreSQL.

Overview

Blender-Final is a full-stack AI application that generates Blender Python scripts from natural language prompts.

Users can enter requests such as:

Create a cube in Blender using BPY

and receive Blender Python code generated by a fine-tuned Qwen 2.5 Coder model.

The application includes:

React-based chat interface
FastAPI backend
PostgreSQL conversation storage
Fine-tuned Qwen 2.5 Coder LoRA model
Local GPU inference using CUDA

Database Setup

Create PostgreSQL database:

CREATE DATABASE blender;

Update connection string inside:

backend/database.py

Example:

engine = create_engine(
    "postgresql://<username>:<password>@localhost:5432/blender"
)
Backend Setup

Navigate to backend:

cd backend

Create virtual environment:

py -3.11 -m venv venv

Activate:

venv\Scripts\activate

Install dependencies:

pip install -r backend_requirements.txt

Install CUDA-enabled PyTorch:

pip uninstall torch torchvision torchaudio -y

pip install torch==2.5.1+cu121 torchvision==0.20.1+cu121 torchaudio==2.5.1+cu121 --index-url https://download.pytorch.org/whl/cu121

Verify CUDA:

python -c "import torch; print(torch.__version__); print(torch.cuda.is_available())"

Expected:

2.5.1+cu121
True

Start backend:

uvicorn main:app --reload --port 8001

Swagger UI:

http://127.0.0.1:8001/docs
Frontend Setup

Navigate to frontend:

cd frontend

Install packages:

npm install

Start frontend:

npm run dev
